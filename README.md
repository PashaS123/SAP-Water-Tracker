# Household Water Use Tracker

Static APES Sustainability Action Project website focused on household water consumption.

## Files

- `index.html` - page content, survey, plan section, and sources
- `styles.css` - responsive visual design
- `app.js` - calculator, live estimate, and personalized recommendations

## How to Open

Open `index.html` in a browser. The site does not need a backend or server.

## Project Focus

The site helps classmates or families estimate household water use from showers, toilets, faucets, dishes, laundry, leaks, and habits. The results section creates a personalized one-week water-saving plan using EPA WaterSense and ENERGY STAR assumptions.
